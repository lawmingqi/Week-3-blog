What I have learned this week

Making Inputs

  • Creates interactive controls for users
  • Different input types: button, checkbox etc

Media Query

  • Key part of a responsive website
  • Uses the @media rule
  • Media features -> test a specific feature of the device
    For example:
    @media (orientation: landscape){}
    @media (width: 480px){ }

  • Min-width<= viewport <= Max-width
    then media query is true

Layout in CSS

  • Static : Default value
  • Relative : element positioned relative to its normal position
  • Absolute : element positioned relative to its first positioned
  • Fixed : element positioned relative to browser window

Flexbox

  • Flexbox container is the parent of the flex item
  • Children of the flexbox container can be laid out horizontally or vertically
  • Using justify-content to space our items within a container

Class

  • The .class selector selects elements with a specific class attribute
    For eg, .class { css declarations; }

    Style all elements with class="hometown":
    p.hometown { background-color: yellow; }

  • Assign name to a HTML element (variables in C#)
  • Classes can be reused and can be stacked
  • The class name is case sensitive!
